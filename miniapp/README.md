# 📱 微信小程序前端

> **状态**：开发中（TODO）

## 📋 目录职责

本目录用于存放微信小程序前端代码，包括：

- **页面开发**：wxml、wxss、页面逻辑
- **组件开发**：自定义小程序组件
- **云调用适配层**：微信云服务接口适配
- **用户界面**：登录、任务管理、结果展示等

## ⚠️ 开发注意事项

### 🔐 安全规范
- ❌ **严禁**在小程序中硬编码API密钥
- ❌ **严禁**将生产环境配置提交到仓库
- ✅ 使用微信云开发的配置管理
- ✅ 敏感信息通过安全接口获取

### 📱 兼容性要求
- 支持微信小程序最新版本
- 适配不同手机屏幕尺寸
- 确保在iOS和Android上表现一致

### 🔄 与后端协作
- 严格遵循API文档规范
- 统一错误处理机制
- 保持与Web前端功能一致
